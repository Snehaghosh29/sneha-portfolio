#  Sneha's Portfolio

Welcome to my personal portfolio repository.  
A modern, responsive, AI-enhanced front-end focused project built to showcase Full Stack + Deep Learning capabilities.

---

## Live Demo  
https://sneha-portfolio-chi-six.vercel.app/

---

##  Key Features  
- Smooth section navigation  
- Premium UI with neon glass aesthetics  
- Image-based fashion recommendation (Visual Product Matcher)  
- Responsive layouts for all screens  
- Built without backend required for contact submissions  

---

## Tech Stack  
- **Frontend:** Next.js, TailwindCSS, Framer Motion, GSAP  
- **Forms:** Web3Forms API (Auto-mail support planned to be added)
